# co-skills

Skills d'agents partagés avec l'équipe de Compagnons.

## Installation

Demandez ceci à votre agent:

> Suivez les instructions dans AGENT_INSTRUCTIONS.md dans ce repo : [https://github.com/compagnonsco/co-skills](https://github.com/compagnonsco/co-skills)

## Mise à jour

Utilisez le skill `/co-agent-update` pour que votre agent mette à jour le repo et synchronise les skills localement.
